# 🎵 Melody Masters Instrument Shop – Online Store

## 📌 Project Description
Melody Masters Instrument Shop is a web-based e-commerce application designed to extend a traditional music instrument store into an online platform. The system allows customers to browse, purchase, and manage musical instruments and digital products while enabling staff and administrators to efficiently manage inventory, orders, and users.

---

## 👥 User Roles
- **Guest**: Browse products and register an account
- **Customer**: Place orders, track purchases, download digital products, and submit reviews
- **Staff**: Manage inventory and process customer orders
- **Administrator**: Full system control including user, product, and order management

---

## 🛒 Key Features
- Product browsing without authentication
- Secure user registration and login
- Can reset password
- Shopping cart and checkout system
- Free shipping for orders over £100
- Digital sheet music downloads after purchase
- Order history and order tracking
- Product reviews for verified purchases
- Inventory and order management dashboard

---

## 🗂️ Product Categories
- Guitars
- Keyboards
- Drums & Percussion
- Wind Instruments
- String Instruments
- Accessories
- Digital Sheet Music

---

## 🗄️ Database Overview
- Users
- Categories
- Products
- Digital downloads
- Orders
- Order Items
- Reviews (verified customers only)

---

## ⚙️ System Rules
- Physical products require shipping
- Free shipping for orders above £100
- Digital products available only after successful payment
- Stock limits enforced
- Reviews allowed only for purchased products

---

## 🖥️ Technical Highlights
- Responsive web design
- Secure authentication and access control
- Session-based shopping cart
- Input validation and error handling
- Scalable relational database design

---

## 🎯 Purpose
This project was developed as an academic assignment to demonstrate the design and implementation of a role-based online shopping system with real-world business rules.
